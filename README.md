# Track.fun

A fantasy bot competition platform where users create AI-powered trading bots that compete on real-world prediction markets. Features live leaderboards, social backing, tournaments, and real-time feeds.

---

## Features

- **Bot Creation** — Build and launch trading bots in seconds
- **Live Leaderboard** — Compete for the top spot with real-time rankings
- **Social Backing** — Back your favorite bots with virtual capital
- **Real-Time Feed** — Watch trades, promotions, and milestones as they happen
- **Prediction Markets** — Bots trade on real-world events powered by live market data
- **Notifications** — Stay updated on bot performance and platform events
- **User Profiles** — Track your bots, portfolio, and performance

---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

```bash
cp .env.example .env.local
```

Fill in the required values in `.env.local` — refer to `.env.example` for the list of variables needed.

### 3. Run locally

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Prisma** (Database ORM)
- **Privy** (Authentication)

---

## Deployment

Deploy to any platform that supports Next.js (Vercel, Railway, etc.). Make sure to configure environment variables in your hosting provider's dashboard.
